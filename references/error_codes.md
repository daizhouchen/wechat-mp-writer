# 微信公众平台 API 常见错误码

## Token 相关

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 40001 | 获取 access_token 时 AppSecret 错误，或者 access_token 无效 | 检查 AppSecret，重新获取 token |
| 40002 | 不合法的凭证类型 | 检查请求参数 |
| 40013 | 不合法的 AppID | 检查 WECHAT_APP_ID 是否正确 |
| 41001 | 缺少 access_token 参数 | 确保请求中包含 token |
| 42001 | access_token 超时（有效期 2 小时） | 刷新 token 后重试 |

## 素材相关

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 40007 | 不合法的媒体文件 ID | 检查 media_id 是否有效 |
| 40009 | 不合法的图片文件大小（限 10MB） | 压缩图片后重新上传 |
| 40010 | 不合法的语音文件大小 | 压缩音频 |
| 40011 | 不合法的视频文件大小 | 压缩视频 |
| 40012 | 不合法的缩略图文件大小 | 调整缩略图尺寸 |
| 40015 | 不合法的 media_type 类型 | 检查 type 参数（image/voice/video/thumb） |
| 45028 | 素材数量达到上限（图片 5000，其他 1000） | 清理旧素材后重试 |

## 草稿与发布

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 45008 | 图文消息超过限制（单次最多 8 篇） | 减少文章数量 |
| 45022 | 标题超过限制（64 字节） | 缩短标题 |
| 45025 | 每日群发数超过限制 | 等待次日再发 |

## 权限与限流

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 45009 | 接口调用超过限制（通常 5000次/天） | 等待次日，或优化调用频率 |
| 48001 | API 功能未授权 | 检查公众号是否有对应权限（认证订阅号/服务号） |
| 48004 | API 接口被封禁 | 联系微信客服 |
| 50001 | 用户未授权该 API | 检查用户是否关注了公众号 |

## 网络与系统

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| -1 | 系统繁忙 | 稍后重试 |
| 40003 | 不合法的 OpenID | 检查 OpenID 格式 |
| 40014 | 不合法的 access_token | 重新获取 token |

## IP 白名单

| 错误码 | 说明 | 处理方式 |
|--------|------|----------|
| 40164 | 调用接口的 IP 地址不在白名单中 | 登录 mp.weixin.qq.com，将机器公网 IP 添加到白名单 |
