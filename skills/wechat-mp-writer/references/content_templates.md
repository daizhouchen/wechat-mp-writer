# 内容类型模板

## 干货教程

```
## 开头：问题引入
描述目标读者遇到的具体问题或困惑，让读者产生"这说的就是我"的代入感。

## 正文：步骤拆解
### 第一步：XXX
具体操作 + 截图/示意图

### 第二步：XXX
具体操作 + 注意事项

### 第三步：XXX
具体操作 + 常见错误提醒

## 实操演示
用一个完整案例把所有步骤串起来，让读者看到从0到1的全过程。

## 总结要点
- 要点1
- 要点2
- 要点3
+ 一句话总结核心方法论
```

## 观点评论

```
## 开头：现象描述
用一个最近的事件/热点/现象引出话题，让读者快速进入语境。

## 亮出观点
用一句话清晰表达你的核心观点（可以适度"反直觉"以引发思考）。

## 论据支撑
### 论据一：数据/案例
### 论据二：逻辑推理
### 论据三：类比/对比

## 引发讨论
抛出一个开放性问题，引导读者在评论区发表看法。
```

## 行业分析

```
## 开头：行业背景
简述行业现状，用1-2个关键数据点抓住注意力。

## 数据解读
### 关键数据1：XXX
解读这个数据背后的含义

### 关键数据2：XXX
与往年/竞品对比

## 趋势判断
基于数据和行业观察，给出3个趋势判断：
1. 短期趋势（3-6个月）
2. 中期趋势（1-2年）
3. 长期方向

## 行动建议
针对目标读者给出具体可执行的建议。
```

## 产品介绍

```
## 开头：痛点场景
描述一个目标用户经常遇到的痛苦场景，引发共鸣。

## 解决方案
引出产品，说明它如何解决上述痛点（注意：先价值后产品，不要一上来就推销）。

## 产品亮点
### 亮点1：XXX（配图/动图）
### 亮点2：XXX
### 亮点3：XXX

## 用户证言（可选）
真实用户的使用反馈。

## 使用引导
如何获取/购买/试用 + CTA
```

## 活动推广

```
## 开头：活动亮点
用最吸引人的一点开场（大咖嘉宾/超值优惠/独家体验）。

## 活动详情
- 时间：
- 地点：
- 参与方式：

## 你能获得什么
列出参与者的具体收获（用数字量化）。

## 限时优惠（如有）
制造紧迫感：名额有限/早鸟价/限时折扣。

## 报名入口
清晰的行动指引。
```

## 人物故事

```
## 开头：人物引入
用一个细节/场景/金句引出主人公，让读者产生好奇。

## 关键转折
讲述改变主人公命运的关键时刻，制造情节张力。

## 成长历程
按时间线或主题线展开故事，穿插细节和对话。

## 启发总结
从个人故事升华到普遍道理，给读者可带走的思考。
```
